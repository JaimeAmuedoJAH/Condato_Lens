## Creación entorno virtual (venv)

```python -m venv venv```

## Activación entorno virtual

```venv/Scripts/activate```

## Instalación dependencias

```pip install -r requirements.txt```

## Crear archivos y carpetas siguientes

- clean_data (datos procesados)
- data (datos crudos, solo filtrados)
- .env (archivo de configuración de la base de datos)


No olvidar eliminar archivos readme.md que esten SOLO dentro de las carpetas.
